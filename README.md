# Data Visualization with Python  

## Overview
This project focuses on **visualizing insights from a cleaned dataset** using Python, with an emphasis on **Matplotlib and Seaborn** for static data visualization.  
The goal is to convert structured data into clear, interpretable charts that reveal trends, patterns, and relationships across variables and demographic groups.

The visualizations are designed to:
- Communicate insights effectively
- Compare values across categories and groups
- Support exploratory and explanatory data analysis

---

## Dataset

- **Source:** Survey or structured dataset  
- **File:** `star_wars_survey_cleaned.csv` 
- **Type:** Cleaned data (categorical, ordinal, numerical, demographic)  
- **Origin:** Output from a prior data wrangling and cleaning process
